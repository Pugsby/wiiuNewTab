## Installation
### Method 1 (Manual)
1. Download the source code
2. Go to chrome://extensions
3. If "Developer Mode" in the top right is disabled, enable it.
4. Click "Download Unpacked"
5. Select the unzipped source code.
### Method 2 (Quick)
1. Download the latest build
2. Go to chrome://extensions
3. Drag the .crx file onto the page
4. It may ask you if the extension is save, click the three dots next to the extension then "Keep"
